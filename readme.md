## Routing

| Method |  Endpoint  |       Query       |     Description      |
|--------|------------|-------------------|----------------------|
| GET    | /          |                   | Root page.           |
| GET    | /hello     | ?name=<username>  | Salutation for user. |
| GET    | /bye       |                   | Say goodbye.         |
| GET    | /ipinfo    |                   | Get IP remote info.  |
| GET    | /secret    |                   | Get random string.   |
| POST   | /data      | any data          | Echo data.           |
